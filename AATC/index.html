Update v0.1.5!

(Pretend this is 0.2.0...)

Additions:

- Added SFX Management!

    SFX:
    • You can swap between managing Music and SFX with the "Switch" button
    • You can preview your SFX by clicking the "Rename" button but not renaming the SFX
    • You will not lose either when switching, and they will both be exported in the same zip file
    • In the future, all different custom assets will be able to be edited within the same session and exported to one zip file
    • Importing a zip will let you choose to start in either sound editing or music editing, unless one of them doesn't exist, in which case the program knows it can just choose the only option.
  • Added the ability to name your zip file before exporting
  • Added a loading animation for importing and exporting
  • Added text that tells you the progress of what you're loading/exporting
    • Example: "Loading Song (3/5)"
  • Added the Switch button
  • Added a sound for selecting from a dropdown menu
  • Rewrote some code

    Bug fixes:

  • Fixed a bug where you could press the Edit and Delete buttons without any files
  • Fixed a bug where you could upload a zip file that had no custom
  • Fixed a bug where text boxes wouldn't show up
  • Fixed a bug where you could upload a second file of the same name, messing up your file
  • Fixed a bug where you could upload nothing
  • Fixed a bug where some internal data would not initialize properly
  • Fixed bugs related to the program improperly accessing files in certain situations

    Changes:

  • Changed the instructions once you export a file
  • Changed exporting to hide the editor GUI
